# dotfiles

OS X / Ubuntu dotfiles.


### Installation

```
$ git clone https://github.com/nakabonne/dotfiles.git
```

## macOS

### deploy

```
$ cd dotfiles
$ sh ./link.sh
```

## Initialize

```
$ cd dotfiles
$ source .bash_profile
$ sh ./install.sh
```
